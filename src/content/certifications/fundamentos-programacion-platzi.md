## Qué aprendí

- Pensamiento lógico y resolución de problemas aplicados a la programación.
- HTML semántico: landmarks, headings, formularios y su impacto en accesibilidad y SEO.
- CSS moderno: modelo de caja, Flexbox, CSS Grid y diseño responsive con media queries.
- Primeros pasos con JavaScript: variables, tipos, control de flujo y manipulación básica del DOM.
- Flujo de trabajo frontend end-to-end: editor, navegador, herramientas de desarrollo y despliegue.
