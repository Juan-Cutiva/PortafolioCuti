## Qué aprendí

- Utility-first CSS y composición de clases con `@apply` y arbitrary values.
- Breakpoints, responsive design y variantes (hover, focus, dark).
- Configuración del design system via `tailwind.config` (colores, tipografía, espaciado).
- Uso correcto de plugins oficiales (forms, typography, aspect-ratio).
