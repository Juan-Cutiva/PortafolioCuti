## Qué aprendí

- Flujos de trabajo con ramas: feature branches, merge vs rebase, cherry-pick.
- Resolución de conflictos y recuperación de historial con reflog.
- Colaboración vía pull requests, code review y protección de ramas.
- Integración con GitHub (Issues, Actions, Projects) y GitHub Pages.
